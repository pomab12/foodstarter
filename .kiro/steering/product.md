# Product Overview

A professional resume website built as a content-driven application. Work experience and education are managed as markdown files with frontmatter, allowing easy updates without code changes.

## Key Features

- Content-driven resume with markdown-based job and education entries
- Interactive skills filter to highlight relevant experience
- Modern UI with shadcn/ui components
- Server-side rendering for optimal performance
- Dark/light theme support

## Content Structure

Jobs and education entries live in `content/` as markdown files with structured frontmatter (dates, tags, company info, etc.). The content-collections library transforms these into typed data at build time.
